## useSceneService

地图场景管理服务

## API

```ts pure
import { useSceneService } from '@antv/dipper';
const { sceneService, scene } = useConfigService();
```

## 参数

无

## 结果

| 参数         | 说明                     | 类型  | 备注 |
| ------------ | ------------------------ | ----- | ---- |
| sceneService | sceneService 实例        |       | -    |
| scene        | Scene 实例，操作地图配置 | `any` | -    |
